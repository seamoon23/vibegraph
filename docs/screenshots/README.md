# 스크린샷 추가 방법

README 상단 표에 들어갈 이미지 3장을 이 폴더에 넣으면 됩니다.

| 파일명 | 내용 | 만드는 법 |
|---|---|---|
| `report.png` | 개별 리포트 | `sample_report.html` 더블클릭 → 브라우저 화면 캡처 |
| `dashboard.png` | 전체 조회 페이지 | `vibe dashboard` 실행 → 열린 `index.html` 캡처 |
| `growth.png` | 성장 리포트 | `vibe growth --all` 실행 → 열린 `growth.html` 캡처 |

## 캡처 팁
- Windows 캡처 도구: `Win + Shift + S`
- 다크 테마라 가로로 넓게(브라우저 창 1000px 이상) 잡으면 깔끔합니다.
- 개인 프로젝트명/내용이 보이면 가리거나, 샘플 데이터로 캡처하세요.

이미지를 넣고 커밋하면 README 상단에 자동으로 표시됩니다.
